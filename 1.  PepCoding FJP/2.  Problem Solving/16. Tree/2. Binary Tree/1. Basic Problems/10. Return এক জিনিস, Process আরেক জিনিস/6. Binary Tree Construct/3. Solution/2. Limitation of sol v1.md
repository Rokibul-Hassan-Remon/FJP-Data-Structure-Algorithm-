### 1️⃣ **Global Static Variable ব্যবহার**

তুমি `preOrderIndx` static member হিসেবে ব্যবহার করেছ state maintain করার জন্য।  
এটা কাজ করে, কিন্তু design-wise clean না। কারণ:

- Static variable recursive function-এর মধ্যে hidden dependency তৈরি করে।
    
- Thread-safe না।
    
- অন্য test case চালালে pre-Order Index carry over হতে পারে।
    

✅ **Fix:** Index টা recursive function-এর parameter হিসেবে pass করা উচিত — বা, reference-like wrapper object ব্যবহার করা যায়।

---

### 2️⃣ **In-order index খোঁজার জন্য Linear Search**

তুমি while loop দিয়ে in-order[]-এ search করছ।  
এটা প্রতিবারে O(n) time নেয়।  
Result → পুরো algorithm হয়ে যায় **O(n²)**.

✅ **Fix:** একবার শুরুতেই hash-map বানাও, যাতে `value → index` map করে রাখো।  
তাহলে O(1)-এ index পেয়ে যাবে।

---
